# 📊 Dashboard Interactivo de Ventas con Streamlit

¡Bienvenido a primer proyecto de Ciencia de Datos desplegado en la nube! Esta aplicación es una herramienta interactiva diseñada para demostrar cómo transformar un archivo CSV estático en un panel de control dinámico utilizando **Python**.

## 🚀 Funcionalidades de la App
- **Carga Automática:** Lee datos directamente desde un archivo `dataset.csv`[cite: 2].
- **Filtros Dinámicos:** Permite al usuario filtrar los datos por categoría desde la barra lateral[cite: 2].
- **Estadísticas en Tiempo Real:** Calcula totales y promedios según la selección del usuario[cite: 2].
- **Visualización Gráfica:** Genera gráficos de barras automáticos con Matplotlib[cite: 2].

## 🛠️ Tecnologías Utilizadas
- **Python:** Lenguaje principal.
- **Streamlit:** Framework para la interfaz web[cite: 2].
- **Pandas:** Manipulación y limpieza de datos[cite: 2].
- **Matplotlib:** Creación de gráficos estadísticos[cite: 2].

## 📂 Estructura del Repositorio
- `app.py`: El código principal de la interfaz[cite: 1, 2].
- `dataset.csv`: El archivo de datos de ejemplo[cite: 1, 2].
- `requirements.txt`: Librerías necesarias para que la app funcione en la nube[cite: 1, 2].
- `test.py`: Pruebas de funcionamiento para asegurar que los datos cargan bien[cite: 1, 2].
